Ansible Role for Atlassian Bitbucket Installation.

## Requirements

This role require Ansible 2.9 or higher.

This role was designed for Ubuntu 16.04/18.04/19.10/20.04 or RHEL/CentOS 7/8 or openSUSE Leap 15.1.
